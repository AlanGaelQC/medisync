## ¿Qué es MediSync?
Sistema digital para modernizar la operación de la Clínica San Ángel. Permite agendar citas, gestionar expedientes médicos, enviar notificaciones automáticas y visualizar métricas administrativas.

---

## Stack Tecnológico

| Capa | Tecnología |
|------|-----------|
| Frontend Web | React |
| Frontend Móvil | React Native (Android) |
| Backend | Node.js + Express |
| Base de datos | MySQL — AWS RDS |
| Autenticación | JWT |
| Almacenamiento | AWS S3 |
| Notificaciones | AWS SNS |
| Despliegue Web | AWS Amplify |
| Despliegue Backend | AWS EC2 |

---

## Estructura de Ramas

- `main` — versión estable y aprobada
- `develop` — integración general del equipo
- `feature/backend` — desarrollo del API
- `feature/web` — desarrollo de la app web
- `feature/mobile` — desarrollo de la app móvil
